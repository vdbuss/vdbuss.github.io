Communication in database and Web querying

LUC Theoretical Computer Science group

Project description

  • database interrogation (computation by sequences of queries)
  • reflective relational machines as a formal model for client-server querying
  • formal models of Web querying and browsing
  • distributed and peer-to-peer databases
  • communication complexity in databases
The project has been funded by a collaborative research grant between Flanders and Poland.

People involved

From the LUC group: Jan Van den Bussche, Marc Spielmann (alumnus), Dirk Leinders.

The project is in collaboration with Jerzy Tyszkiewicz and Michal Bielecki from Warsaw University.

System prototype

A peer-to-peer system for Web querying.
