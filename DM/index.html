Data Mining

Theoretical Computer Science Group, Hasselt University

Description

Our research is mainly focused on the following topics
  • improving rule mining algorithms
  • exploiting constraints within rule mining algorithms
  • conjunctive queries as patterns
  • inductive databases
  • graph mining
